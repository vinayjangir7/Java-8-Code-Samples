
# Java 8 Concepts

This repo consists of examples which helps get to know newly added java concepts.



## Topics Covered

- Streams
    - Intermediate Operations
    - Terminal Operations
- Lambda Expression
- Functional Interfaces
    - Predicate
    - Consumer
- Method Reference
- Collectors
    - groupingBy
    - partitioningBy
    - summarizingInt
- Custom Collector
- Parallel Streams
- Split Iterator
- Date Time API
- Optionals
- Fork/Join Framework
